# 用户服务
提供用户信息的增删改查接口

### 查询用户列表
根据条件查询用户分页信息

####  请求URL
```
/users/queryUserPage
```

####  请求方式
- POST

#### 请求参数
|                   参数名                   |类型|必填|说明|默认值|
|:-------|:-------:|:-------:|:-------:|:-------:|
|userName|string|是|姓名||
|phoneNo|string|否|手机号||
|age|integer|是|年龄|28|
